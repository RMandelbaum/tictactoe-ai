# Tic Tac Toe
A CLI game of Tic Tac Toe. Play with two people, against a computer, or watch two computers.


## Installation

$ git clone https://github.com/RMandelbaum/coding_challenges/tree/code-submission

$ cd coding_challenges/ruby

$ bundle install

$ ruby bin/tictactoe
